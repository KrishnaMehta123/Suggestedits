# Overview

The Dashboard → Uninstall View presents data on the number of users uninstalling your app, the number of new activations (first time app launches), and the net number of new installs i.e. new installs – total uninstalls, for specific date ranges.

Please note the following:
  * You must explicitly enable Uninstall detection feature from your dashboard.
  * Uninstall detection is only available for the iOS and Android apps. Uninstall data is not available for Windows apps.
