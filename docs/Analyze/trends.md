# Bookmarks

You can bookmark any funnel, cohort or event query and save it to quickly retrieve later. Bookmarks are specific to each logged in user. No other dashboard users will see the bookmarks you save. 

Bookmarks are convenient for recalling a specific analysis that you want to see frequently. For\
example you can bookmark you important conversion funnels and view them each day or week to see how they are changing over time.
