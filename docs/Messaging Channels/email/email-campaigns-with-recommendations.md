## Email campaigns with recommendations

You can create email campaigns with recommended content. The steps to invoke recommendations within the email messages are explained below. Beyond these, all the steps are the same as the generic ones described in the sections above. 

You can create emails using:

* Custom HTML
* Templates

<Image title="Recommendation Email_1.png" alt={1437} className="border" border={true} src="https://files.readme.io/e727775-Recommendation_Email_1.png" />

## Recommendations in custom HTML emails

You can create personalized recommendations within custom HTML emails in the following manner:

### Link with recommendations

You can add recommendations within links in the following way:

Step 1: Add a link with to your email message

<Image title="Recommendation Email_2.png" alt={1367} className="border" border={true} src="https://files.readme.io/dc8d9f2-Recommendation_Email_2.png" />

Step 2: Personalize the links using recommendations

You can access the recommendations within the 'Display text' and 'URL' fields.You can personalize communication to your users using these fields.

<Image title="Recommendation Email_3 copy.png" alt={676} className="border" border={true} src="https://files.readme.io/fddcf76-Recommendation_Email_3_copy.png" />

### Image with recommendations

Step 1: Add an image to your email message

<Image title="Recommendation Email_4.png" alt={1358} className="border" border={true} src="https://files.readme.io/dd86134-Recommendation_Email_4.png" />

Step 2: Personalize the image using recommendations

You can access the recommendations within the 'URL', 'Alternative Text' and 'Target URL' fields. You can personalize communication to your users using these fields. 

<Image title="Recommendation Email_5 copy.png" alt={678} className="border" border={true} src="https://files.readme.io/a5de6af-Recommendation_Email_5_copy.png" />

The steps further are the same as any other campaigns, as explained as follows:\
[https://docs.clevertap.com/docs/create-campaigns-using-recommendations](https://docs.clevertap.com/docs/create-campaigns-using-recommendations)
