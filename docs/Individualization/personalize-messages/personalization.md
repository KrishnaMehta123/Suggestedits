Profile and event @ personalization

Overview\
Marketers can use Profile and Event personalization using the “@” symbol wherever Profile or Event related information can be input.

Push Notifications\
Profile personalization can be used in the Title and Message fields

InApp Notifications\
Event & Profile personalization can be used in the Title, Body of InApp Notifications.

SMS\
Personalization can be used in the Message field of an SMS

App Inbox\
Profile personalisation can be used in Title and Message fields of App Inbox

Native Display\
Event and Profile Personalization can be used in the Value fields of Native Display campaigns

Email\
Profile personalization can be used in Email body
