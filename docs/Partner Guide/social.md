CleverTap offers access to WhatsApp Business API directly from its dashboard. Besides, if you already have access to WhatsApp Business API via third-party providers such as Exotel, Gupshup, Nexmo, ValueFirst, etc, you can also set up these providers in the CleverTap dashboard and use WhatsApp as a communication channel via these providers.

To get started with WhatsApp, please contact [sales@clevertap.com](mailto:sales@clevertap.com).
