# Overview

[LiveClicker](https://www.liveclicker.com/) is a real-time email personalization platform that empowers marketers to deliver dynamic and interactive content directly within emails. It allows you to modify email elements even after deliveryâ€”ensuring content is always relevant at the moment of open.

By integrating LiveClicker with CleverTap, you can embed personalized timers, images, or offers that automatically update based on user behavior and real-time data. This integration helps marketing teams drive engagement and conversions by adding contextual urgency and relevance to every campaign.

For example:

- **Countdown Promotions:** Add a LiveTimer countdown to sales or limited-time events to create urgency.
- **Dynamic Offers:** Display personalized discounts based on user location or time of open.
- **Event Reminders:** Update content dynamically until the event start time.

***

Consider an example where an email campaign promotes a 24-hour flash sale. Using the **CleverTap \<> LiveClicker** integration, marketers can embed a countdown timer that updates in real-time for each recipient, showing the exact time left when the email is opened.

This creates urgency and improves conversion rates by motivating users to act before the timer expires.

# Prerequisites for Integration

Before you begin, ensure you have the following:

- A valid **LiveClicker account** with campaign creation access.
- A **CleverTap account** with permissions to create Email or In-App campaigns.
- Basic understanding of HTML and embed code usage.

# Integrating LiveClicker with CleverTap

The integration process involves two key steps:

1. [Create LiveTimer Content in LiveClicker](doc:liveclicker#create-livetimer-content)
2. [Embed the Timer Code in CleverTap Campaigns](doc:liveclicker#embed-timer-in-clevertap)

## Create LiveTimer Content in LiveClicker

To create your countdown timer in LiveClicker:

1. Log in to your **LiveClicker** dashboard.
2. Click **Create Campaign** and enter a name for your campaign.
3. On the setup page, select the **LiveTimer** checkbox to add a timer wizard.
4. Configure the following:

   - **Basic Settings:** Choose the campaign name and optional clickthrough URL.
   - **Layout:**

     - _Always show running seconds_ (days, hours, minutes, seconds), or
     - _Only show running seconds in the last 99 hours_ (hours, minutes, seconds).

## Embed the Timer Code in CleverTap

You can embed the LiveClicker code into **Email** or **In-App Message** campaigns.

### Embed in Email Campaigns

1. In your CleverTap dashboard, create a new **Email Campaign**.
2. In the **What** section, select an email template.
3. Click **Source** to view the HTML editor.
4. Paste the **LiveClicker embed code** inside the `<body>` tag.
5. Save and preview the campaign to confirm proper display.

### Embed in In-App Messages

1. Create a new **In-App Message Campaign** in CleverTap.
2. In the **What** section, select **Custom HTML Template**.
3. Paste the **LiveClicker embed code** in the custom HTML block.
4. Test, preview, and publish the campaign.

# Verify Integration

After publishing, test your campaign to ensure the timer renders correctly and updates dynamically.

### Verify in Email

1. Send a test email to your inbox.
2. Open the email and confirm that:

   - The countdown timer loads instantly.
   - The timer reflects the correct time remaining.
   - The expiration image appears once the timer reaches zero.

### Verify in LiveClicker Dashboard

1. In your LiveClicker campaign view, check for incoming opens.
2. Confirm that timer impressions and clickthroughs are logged as expected.

# Troubleshooting

| Issue                      | Possible Cause                  | Recommended Fix                                                      |
| -------------------------- | ------------------------------- | -------------------------------------------------------------------- |
| Timer not visible in email | Embed code not placed correctly | Ensure the LiveClicker code is inside the `<body>` tag.              |
| Timer shows zeros          | Expiration image not configured | Upload a valid expiration image in LiveClicker campaign settings.    |
| Timer not updating         | Email client cached the image   | Reopen the email after a few minutes or disable caching for testing. |
