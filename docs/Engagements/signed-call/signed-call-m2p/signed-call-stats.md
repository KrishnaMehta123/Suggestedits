After publishing the Signed Call™ campaigns, you can view the statistics from the dashboard. Navigate to *Campaigns* and select the specific campaign from the campaign list. To filter out Signed Call specific campaigns, apply the channel filter. 

<Image alt="Apply Campaign  Filter" align="center" border={true} src="https://files.readme.io/8bf3f1c-filter.png">
  Apply Campaign  Filter
</Image>

Based on the type of campaign you select (Audio or Interactive Voice campaign), a stats page opens up where you can view the total *Calls*, *Received* calls, *Answered* calls, and percentage of  *IVR interactions.*

<Image alt="Stats view for an Interactive Voice Response Campaign." align="center" border={true} src="https://files.readme.io/fbafe0a-Stats_1.png">
  Stats view for an Interactive Voice Response Campaign.
</Image>

* **Calls**: Represents the number of calls made across the target users
* **Received**: Represents the number of calls landed on the end user device
* **Answered**: Represents the total number of calls answered by the end users
* **IVR Interactions**: Represents the overall audio interactions by the end users

For the *Users drop off time* and *Response Received* events, you also get a bar chart view (as shown above) that helps you analyze the call data
