# Set Up Signed Call Integration

The Signed Call feature is compatible with Android, iOS, and Web platforms. Select and integrate Signed Call based on your specific platform requirements. From the CleverTap dashboard, perform the following steps:

1. Navigate to *Settings* > *Channels* >  *Signed Call* 

<Image title="Signed Call Integration" alt="A dashboard screen showing Signed Call SDK Integration steps for Android, iOS, and Web platforms. Refer to the Quick Start Guide for documentation" align="center" border={true} src="https://files.readme.io/40ea36f-Signed_Call_settings.png">
  Signed Call Settings
</Image>

2. Choose the integration platform and refer to the respective *Quickstart Guide* to integrate:

* [Signed Call Android SDK](https://developer.clevertap.com/docs/signed-call-android-sdk)
* [Signed Call iOS SDK](https://developer.clevertap.com/docs/signed-call-ios-sdk)
* [Signed Call Web SDK](https://developer.clevertap.com/docs/signed-call-web-sdk)

# Set Up Signed Call Branding

You can customize the brand logo, theme, and color for both Signed Call campaigns:

* *Machine to Person*
* *Person to Person*  

An incoming call with business branding increases call credibility and personalizes the user experience. 

This section explains how to set up *Content*, and *Style* for the incoming and ongoing call screens. 

To set up Signed Call Branding for your *Machine to Person* or *Person to Person Calling* campaign:

1. Navigate to **Settings** > **Channels** >**Signed Call**.

<Image alt="Signed Call Branding" align="center" border={true} src="https://files.readme.io/09da307-Signed_Call_Campaign.png">
  Signed Call Branding
</Image>

2. Navigate to the **Branding** tab.
3. Basis the campaign type (Machine to Person or Person to Person) you want to configure, click **Edit**.
4. Enter the *Image URL* of your **Brand Logo**. CleverTap fetches your brand image from the provided URL or your CDN and displays it during the call.
5. Click **Done**.

<Image alt="Define Brand Logo" align="center" border={true} src="https://files.readme.io/04889ce-branding_gif.gif">
  Define Brand Logo
</Image>

2. Click the **Style** tab to define custom styling for the *Incoming* and *Ongoing* call screens.
3. Select the **Background Color**.
4. Select the color of the text.
5. Select **Button Theme**(Black or White).
6. Click **Save**.

<Image alt="Signed Call Campaign Styling" align="center" border={true} src="https://files.readme.io/a522ff1-SC_Styling.png">
  Signed Call Campaign Styling
</Image>

You can preview how the call screen will look on different devices by switching between the Android and iOS toggle available on the right-hand side. Refer to the following image for a better understanding:
