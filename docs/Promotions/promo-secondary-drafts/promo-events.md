# Overview

Promo events are system events generated by CleverTap's promo module in response to reward actions such as issuing a coupon, crediting wallet points, or distributing a partner voucher. These events come enriched with event metadata that can be used to personalize communication campaigns using Liquid tags.

These events help power dynamic user engagement workflows. For example, a user rewarded with a coupon can be automatically added to a Live segment that receives a push notification with the coupon code dynamically inserted into the message.

# Promo System Events

| Reward Module       | Event Name                  | Description                                   | Trigger Condition                                  | Event Metadata                                                                                                                                                          |
| ------------------- | --------------------------- | --------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Coupon**          | CT Coupon Rewarded          | User is rewarded with a coupon                | When the coupon is issued through a promo campaign | `Coupon Name`, `Coupon Id`, `Coupon Code`, `Campaign Id`, `Description`, `Terms and Condition`, `Coupon Start Date`, `Redemption Activation Date`, `Coupon Expiry Date` |
|                     | CT Coupon Redeemed          | Coupon is redeemed through the redemption API | When user redeems the coupon                       | `Order Id`, `Coupon Code`, `Coupon Id`, `Coupon Name`, `Discount Amount`, `Cashback Points`                                                                             |
|                     | CT Coupon Reverted          | A redeemed coupon is reversed                 | When redemption is reverted                        | Same as Coupon Redeemed                                                                                                                                                 |
| **Wallets**         | CT Points Credited          | Points added to the user's loyalty wallet     | API/manual/campaign/cashback-based point credit    | `Wallet Id`, `Campaign Id`, `Source`, `Wallet Description`, `Points`, `Transaction Id`                                                                                  |
|                     | CT Points Debited           | Points deducted from the wallet               | API/manual point debit                             | `Wallet Id`, `Source`, `Wallet Description`, `Points`, `Transaction Id`                                                                                                 |
| **Partner Voucher** | CT Partner Voucher Rewarded | User receives a third-party/partner voucher   | Rewarded via campaign or voucher API               | `Code Name`, `Voucher List Id`, `Code Description`, `Code Provider`, `Code Snippet Name`, `Campaign Id`, `Source`                                                       |
