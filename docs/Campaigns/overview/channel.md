- In-app
- SMS
- App Inbox
- Native display

## Desktop/Mobile Web Channels

- Web Pop-up
- Web Exit-intent

## Email Channels
- Email

## Social Channels
- WhatsApp

## Remarketing Channels

- Google Ads
- Server Channels
- Webhooks
