In addition to the [onboarding glossary](https://docs.clevertap.com/docs/glossary-2), this list of definitions provides more information on common analytics terms.

##A
###Acquired Users
New users who have downloaded an app.

###AIC Framework
An acknowledgment, interest, and conversion framework that provides an understanding of engagement through three layers of increasing engagement value during a customer lifecycle.

###AARRR Framework
An acquisition, activation, retention, referral, and revenue framework that provides an understanding of customer behavior in progressive stages during a customer lifecycle.

###Active Event
An [event](https://docs.clevertap.com/docs/events) that is currently logging data.

###Advanced Role-based Access Control
The ability for admins to create [custom roles](https://docs.clevertap.com/docs/role-based-access-control#section-custom-roles) that grant access to user groups based on specific permission.

###Anonymous Users
A user who has [no personal identifier](https://docs.clevertap.com/docs/user-profiles#section-user-profile-types) in CleverTap.

###Daily active users (DAUs)
