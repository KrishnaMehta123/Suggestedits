# Overview

The Find People View lets you segment your users by actions taken, actions not taken, or user profile properties that match a set of criteria you have defined.

<Image title="Segments_Find_users_search.png" alt={1289} className="border" width="smart" border={true} src="https://files.readme.io/3574172-Segments_Find_users_search.png" />

You can view detailed engagement statistics, demographic, device, and location information for those users. Inside your user segments, you can drill down into individual user profiles to see their detailed user profile and activity data.

<Image title="Segments_Find_users_search_results.png" alt={1276} className="border" border={true} src="https://files.readme.io/a01df2a-Segments_Find_users_search_results.png" />

You can also run targeted push, in-app, email, and web campaigns for these user segments from the Find People view.

## Download User List

After segmenting the users, in your desired format, you may also download the user list in CSV format. 

<Image title="Download_Users.jpg" alt={418} className="border" width="80%" border={true} src="https://files.readme.io/73ce4ee-Download_Users.jpg" />

You may select the required attributes for the user list.

<Image title="Download_pref.jpg" alt={502} className="border" width="80%" border={true} src="https://files.readme.io/1555218-Download_pref.jpg" />

The CSV file contains details only about the users who performed the selected event and have any of the selected information available.
