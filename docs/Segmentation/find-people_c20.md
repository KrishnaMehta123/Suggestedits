# Overview

The Find People View lets you segment your users by actions taken, actions not taken, or user profile properties that match a set of criteria youâ€™ve defined.

<Image title="Segments_Find_users_search.png" alt={1289} className="border" border={true} src="https://files.readme.io/3574172-Segments_Find_users_search.png" />

You can view detailed engagement statistics, demographic, device, and location information for those users. Inside your user segments, you can drill down into individual user profiles to see their detailed user profile and activity data.

<Image title="Segments_Find_users_search_results.png" alt={1276} className="border" border={true} src="https://files.readme.io/a01df2a-Segments_Find_users_search_results.png" />

You can also can run targeted push, in-app, email, and web campaigns for these user segments from the Find People view.
