# General Considerations

These are global considerations across CleverTap. 

## Supported Browsers

Google Chrome version 76 and higher.

# Special Considerations

## Event and Property Names

It is a good practise to avoid the following characters in event and property names:

* Dot(.) 
* Colon(;) 
* Collar sign($) 
* Single quote(')
* Double quote(")
* Backslash(/)
