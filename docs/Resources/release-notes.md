# 2018

## Journeys (Beta) - Visual Campaign Building

With Journeys you can build rich, omni-channel campaigns for every life-cycle stage of your users. Drag and drop editing lets you quickly compile and visualize long running campaigns that can span multiple marketing channels so you can optimize user interactions. 

Journeys is a premium feature that will be available to all accounts while in Beta.
