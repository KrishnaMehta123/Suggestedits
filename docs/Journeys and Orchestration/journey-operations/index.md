# Manage Journeys

1. Select **Journeys**. This takes you to the *All Journeys* page.
2. Select a journey's checkbox to manage its functioning.

<Image title="View and manage Journeys on the All Journeys page" alt={1345} align="center" width="90% " border={true} src="https://files.readme.io/68d56f51e3beac7e531019a535223e3b6b0534cc99cc03d30207e2ea6a98c72a-Manage_Journeys.png">
  Manage Journeys
</Image>

This activates the *Action Buttons* displayed on the top left. The buttons are:

<Table align={["left","left"]}>
  <thead>
    <tr>
      <th>
        Action Buttons
      </th>

      <th>
        Description
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        * \*Pause and Resume\*\*![Pause\_Resume](https://files.readme.io/70f33f0-Pause_Resume.png)
      </td>

      <td>
        Temporarily halts the qualification of new users in the Journey. Users already in the journey will continue traveling through the journey or receiving messages. You can select more than one Journey to pause or resume them altogether.
      </td>
    </tr>

    <tr>
      <td>
        * \*Stop\*\*![Stop](https://files.readme.io/e99e396-Stop.png)
      </td>

      <td>
        Completely halts the journey. It stops:  

        * qualification of new users
        * movement of users within the journey
        * delivery of campaigns.**NOTE**: Restarting a stopped journey is not possible. You can select more than one Journey to stop them together.
      </td>
    </tr>

    <tr>
      <td>
        **Ellipsis![Ellipsis](https://files.readme.io/38242f5-ellipsis.png)**
      </td>

      <td>
        More actions are available from the ellipsis icon:  

        * **Clone**: Enables you to copy and create a new journey. You may use the cloned journey to create an edited version from the base cloned version.
        * **Edit**: Enables you to edit the journey to tweak for any enhancements.
      </td>
    </tr>
  </tbody>
</Table>
