# Overview
The Stats for an IntelliNode provide a summary of results collected for each path. This helps in understanding the goal conversions for the paths to make a better decision on path distributions. 

## Node Stats
1. From the *All Journeys* page, click the ellipsis for the required Journey. 
2. Click *View Report > Node Stats*. The stats display for the Journey nodes.
