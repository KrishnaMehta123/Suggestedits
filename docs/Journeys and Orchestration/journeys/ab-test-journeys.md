# Overview

The Stats for an A/B test provide a summary of results collected for each path. This helps in understanding the goal conversions for the paths. 

## Stats

1. From the *All Journeys* page, click. the ellipsis for the required Journey. 
2. Click *View Report > Node Stats*. The stats display for the Journey nodes.

<Image title="Journeys_splitter_node_stats.png" alt={954} className="border" width="80%" border={true} src="https://files.readme.io/35ec484-Journeys_splitter_node_stats.png" />

## A/B Test Stats

1. Click the A/B test to view goal conversion stats for the paths deployed.

<Image title="Goal_Conversion_Stats.jpg" alt={383} className="border" width="80%" border={true} src="https://files.readme.io/391a1c6-Goal_Conversion_Stats.jpg" />

2. Select a goal to view its conversion chart and user distribution stats.\
   Scroll down to view the respective charts.
