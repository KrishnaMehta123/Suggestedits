## What is the image dimension of sending notifications via Web Pop Up? 

The Dimensions and format, which can be used along with the image are mentioned below:\
URL must begin with "https\://"\
The image ratio should be 4:3.\
The maximum frame size is defined for the interstitial pop-up banner and will be 1500 \* 900.\
Formats supported are JPG, JPEG, and PNG.
