Segment, a Customer Data Platform service provider, provides a service that simplifies collecting and using data from the users of your digital properties (websites, apps, etc.). With Segment, you can collect, transform, send, and archive your first-party customer data.

This integration provides the capability to export CleverTap data to [Segment](https://segment.com/).
