After publishing the Campaign, view the statistics from the dashboard.\
Click **Campaigns** > select the specific webhook campaign from the campaign list.\
A Stats page opens up where you can view the overall conversion performance and errors.

<Image title="Webhooks Stats.png" alt={5760} className="border" border={true} src="https://files.readme.io/fbe4e3c-Webhooks_Stats.png" />

The Stats page shows the number of *Qualified Users*:

* **Sent** - This shows the number of users that the Campaign was sent to.
* **Converted users** - This shows the number of users that met the Campaign goal, along with the conversion percentage.
* **Conversion performance**  - This shows the campaign performance.
* **Revenue performance** - This shows the incremental revenue and its percentage increase.
