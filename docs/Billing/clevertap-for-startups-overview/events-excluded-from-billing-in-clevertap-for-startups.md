# Overview

Some system events are excluded from the MAU and Data Point calculations for CleverTap for Startups users.

# Events Excluded from MAU Calculations

The following is the list of events that are excluded from the MAU calculation:

* Notification Sent
* Notification Delivered
* Push Impressions
* App Uninstalled
* Session Concluded
* Reply Sent
* State Transitioned
* Geocluster Entered
* Geocluster Exited 
* AB Experiment Rendered
* AB Experiment Rolled Out
* AB Experiment Stopped
* AB Experiment Disqualified
* Identity Set
* Identity Error
* Identity Reset
* Reachable By
* Partner Sync

# Events Excluded from Data Point Calculations

The following is the list of events that are excluded from the data point calculation:

* App Launched
* UTM Visited
* Notification Delivered
* Notification Sent
* Notification Viewed
* Notification Clicked
* Notification Replied
* App Upgraded
* App Installed
* App Uninstalled 
* Session Concluded
* Reply Sent
* State Transitioned
* Geocluster Entered
* Geocluster Exited
* AB Experiment Rendered
* AB Experiment Rolled Out
* AB Experiment Stopped
* AB Experiment Disqualified
* Identity Set
* Identity Error
* Identity Reset
* Reachable By 

For more information about each of these events, refer to [System Events](https://docs.clevertap.com/docs/events#system-events) and [Debug Events](https://docs.clevertap.com/docs/events#debug-events).
