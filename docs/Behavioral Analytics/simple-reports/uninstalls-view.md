# Overview

The *Uninstall* dashboard presents data on the number of users uninstalling your app, the number of new activations (first-time app launches), and the net number of new installs (i.e., new installs and total uninstalls for specific date ranges).

![1719](https://files.readme.io/07773d4-f469e26-Analytics_View_Uninstall.png "f469e26-Analytics_View_Uninstall.png")

To get to the uninstall view, navigate to *Boards* > *Uninstall*.
