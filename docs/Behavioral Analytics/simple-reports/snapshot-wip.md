# Overview

Snapshot view combines the Today, Mobile App, and Uninstall board. It provides a quick view of mobile app usage, active users, and engaged users across all platforms (Android, iOS, and Web). 

This is the default view.

<Image title="This is hover text" alt="This is alt text for the screen reader" width="80%" border={true} src="https://files.readme.io/121bf32-Snapshot_Boards_Main.png">
  This is an image caption.
</Image>

For this view, navigate to *Boards* > *Snapshot*.

This view displays the following:

* [Mobile App Users](doc:snapshot-wip#mobile-app-users)
* [Today's Active Users](doc:snapshot-wip#todays-active-users)

# Mobile App Users

This section shows the following statistics:

* **Daily Active Users**: Displays the number of users who launched the mobile application today and yesterday. 
* **Weekly Active Users**: Displays the number of users who launched the mobile application in the current and previous week.
* **Monthly Active Users**: Displays the number of users who launched the mobile application in the current and previous month.
* **Total Active Users**: Displays the total active users for the mobile application in the last 30 days.
* **New Activations**: Displays the number of new users who have signed up for the mobile application in a given time frame. 
* **App Uninstalls**: Displays the number of users who have installed the mobile application in a given time frame. 

# Today's Active Users

This section displays the following two graphs:

## User Activity

Displays the number of users who have performed any activity on the app or website. You can also view the number of users who performed any activity on the application or website yesterday and a week ago.

![1134](https://files.readme.io/16bffc2-User_Activity.png "User Activity.png")

## New vs Returning Users

Represents first-time users in your application or website on that day. The *Existing Users* represent users returning to your application or website that day.
