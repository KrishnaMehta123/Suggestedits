## Overview

The flows tab within funnels visually displays the successive steps performed by the users for selected events. 

![1237](https://files.readme.io/6b5a6e4-Flows_tab.png "Flows tab.png")

# Example

For example, if you create a funnel of *Added to cart* and *Charged* events, the flows tab in *Funnels* shows the number of times that the user performed *Added to cart* and *Charged* events based on the interaction depth selected.
