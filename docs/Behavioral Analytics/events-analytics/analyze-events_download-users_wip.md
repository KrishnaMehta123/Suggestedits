# Overview

This section covers how to analyze an event and its various components.

# Analyze an Event

To analyze an event, perform the following steps:

1. From the dashboard, navigate to *Analytics* > *Events*. 
2. Filter by segments and select the event. 
3. If required, filter further by segment behavior. 
4. Click **View details**.

<Image title="Events_Analyze.png" alt={831} className="border" width="80%" border={true} src="https://files.readme.io/6ee5724-Events_Analyze.png" />

# Quick View

The *Quick view* displays all the basic details such as the number of events, the number of profiles, demographics, and sample profiles.

<Image title="Events_Analyze_Quick_View.png" alt={883} className="border" width="80%" border={true} src="https://files.readme.io/ea81958-Events_Analyze_Quick_View.png" />

# Trends and Properties

In the *Trend and Properties* tab, you can analyze the events and people on a daily, weekly, and monthly duration.

In addition, you can further evaluate the selected event by its property and also view the frequency histogram chart for the same event.

# Session

We divide a 24-hour day cycle into 12 buckets of two hours each. The event is analyzed by the time of the day based on these two-hour divisions. 

<Image title="Events_Analyze_Sessions Tab.png" alt={886} className="border" width="80%" border={true} src="https://files.readme.io/06aba6a-Events_Analyze_Sessions_Tab.png" />

# Geography

The event data is displayed by cities, regions, and countries.

# Technographics

You can also analyze the event based on the browsers, operating systems, and devices. Select the relevant options from the *By app field* list including OS version, app version, make, and model.

# Download User List

On filtering the users for your desired events, you may download the user list in the *CSV* format. 

<Image title="Download_Users.jpg" alt={418} className="border" width="80%" border={true} src="https://files.readme.io/ac13c2a-Download_Users.jpg" />

Select the required attributes for the user list.

<Image title="Download_pref.jpg" alt={502} className="border" width="80%" border={true} src="https://files.readme.io/bff17e3-Download_pref.jpg" />

The CSV file contains details only about the users who performed the selected event and have any of the selected information available.
