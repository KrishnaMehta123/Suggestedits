# Overview

Marketing collaterals extensively use image files to add imagery and communicate brand voice. The *Files* section provides a centralized way of storing, uploading, deleting, and organizing files. You can save and reuse the image assets in the *Files* section.

Navigate to *Content* > *Content Manager* > *Files*. The *Files* window displays. 

<Image alt="Files Editor" align="center" border={true} src="https://files.readme.io/f32b6f9-CMS_Files_Main.jpg">
  Files Editor
</Image>

You can select the card view or the list view.  The Files editor offers multiple options such as:

* *Select Views*: Switch between list ( ![](https://files.readme.io/a04f2dd9378d1aeb718ab222636c4ab556a20bfa9a81a5889d37035f32574653-ac80074-list_view_icon.jpg) ) and card ( ![](https://files.readme.io/ea4348c14cb04a42aa837d23e981f39f1a0ba9a20f770d2905a3e736f32ff20c-40d9260-tile_view_icon.jpg) ) view for the image files.
* *Search Files*: Search an image by its name.
* *Sort*: Sort by file name, created date, and last modified date.
* *Multi-Select*: Select multiple files by clicking the **Selection mode** button. 
* *Bulk Delete*: Select multiple files and click the ![](https://files.readme.io/ab2a5cfff2c971a1df4c6a096b5c83eab65fcdbd3175bb53a5b6420d9bf0c9d4-70a339a-delete_all_icon.jpg) icon.

# File Specifications

Listed below are the file specifications you must remember before uploading the files.

| File Type | File Format Supported          | Maximum Size |
| :-------- | :----------------------------- | :----------- |
| Image     | .png, .jpeg, .jpg, .webp, .gif | 1 MB         |
| Video     | .mp4                           | 50 MB        |
| Document  | .pdf                           | 2 MB         |

> ðŸ“˜ File names
>
> File names can include letters (A-Z, a-z), numbers (0-9), underscores (\_), periods (.), hyphens (-), parentheses (()), and spaces. However, names must not begin with a special character. Additionally, the File name must be unique.

# Upload Files

Upload image files to use later in your messaging. 

1. Navigate to *Content* > *Content Manager* > *Files*.
2. Drag and drop the files in the editor. OR\
   You can also click  **Upload Files**  and select the files to upload.

# File Operations

You can click the ![](https://files.readme.io/a82e9ecaffadee9ef9d5a608e9908b62351320f15299c9874c59efe1abfa158b-Ellipses_icon.png) icon from the content card for additional actions to manage files.

<Image alt="File Options" align="center" border={true} src="https://files.readme.io/b218168-CMS_Files_Options.jpg">
  Additional Files Actions
</Image>

You can select from the following options:

* *Copy link*: Click to copy the file link.  
* *Rename*: Click to change the name of the file. 
* *Delete*: Click to delete the file. A file deleted from the *File* section does not affect the files in a running campaign.

## Filter Files

You can filter from the saved files to find the required file quickly. To filter the files:

1. Navigate to  *Content* > *Content Manager* > *Files*.
2. Click **Filters** and apply the required criteria.

<Image align="center" className="border" border={true} src="https://files.readme.io/392fa12-Filter_Files.jpg" />

You can filter files with the following:

* *Time Period*: Search by the time period when the file was created.
* *Type*: Search by the file type.
* *Labels*: Search the file by the assigned label. 
* *Created by*: Search by the person who created the file.

# Organize Files using Folders

> ðŸ“˜ Private Beta
>
> Currently, the Folders in Content Manager is released in Private Beta. If you want to access this feature, contact your Customer Success Manager (CSM).

Organizing files using folders helps maintain a structured system, making locating and managing content easy. By grouping related files into designated folders, you improve accessibility and ensure efficient file management. You can organize files in a folder or using nested folders. You can also move or copy files to the required folder. 

## Create File Folders

To create new *File* folders, follow these steps:

1. Navigate to *Content Manager* > *Files* and click **Create Folder**. 
2. Enter a name for your folder and click **Create**. 

<Image alt="Create and name a folder" align="center" border={true} src="https://files.readme.io/32c330635e1775817715d40d5a8b47e674ace00e0c2f2bf10aace99a67be545b-image.png">
  Create and name a Folder
</Image>

You can also create a nested folder (a folder within a folder) using the same steps. Up to **ten levels** of nested folders are supported, giving you the flexibility to create deep folder structures for comprehensive file organization.

## Move or Copy Files to Folders

To move or copy Files to your folders:

1. Navigate to *Content Manager* > *Files*. The folders are displayed.
2. Double-click to open a folder. The files are displayed.
3. Click **Selection mode** to select multiple files. 
4. Select the required files and click **Organize**.
5. Click **Copy** or **Move** icon. The list of destination folders is displayed. 
6. Select the required folder and click the **Copy** or **Move Here** button at the bottom of the window. 

The files are copied or moved to the destination folders. 

## Navigate and Search Files in Folders

Your files are saved in the respective folders. To search them in folders:

1. Go to *Content Manager* > *Files*. The folders are displayed with the path. 
2. Double-click to open a folder and view the files. You can go back and forth by clicking the folder name in the breadcrumb path. 
3. Use the search box to find a specific file or expand the pane to filter with specific criteria. 

<Image alt="Search for Files or Folders" align="center" border={true} src="https://files.readme.io/b784caccc05f6cd7315fa5e0aec3b1b04dc475387a7aece96edb38344beafd5e-image.png">
  Search for Files or Folders

## Folder Operations

You can perform various operations on your folder from the Files Window.

1. Go to *Content Manager* > *Files*. 
2. Click the Folder card to perform various folder operations.

<Image alt="folder operations" align="center" border={true} src="https://files.readme.io/3fb5d32fa7375dfe18403edee0645fbbbba63ae2a237c694ac1db17d102f6c24-image.png">
  Folder Operations
</Image>

3. Select from the following options:

* *Rename*: Select to rename the folder.
* *Delete*: Select to delete the folder. All files and folders stored within this folder will be deleted.
