# FAQs

## Does CleverTap offer integration with Mixpanel and Appsflyer?

 A. CleverTap offers integration with Mixpanel and Appsflyer. For more details on the list of partners integration, refer to [Pertners Integration.](https://docs.clevertap.com/docs/clevertap-for-startups-integrations#technology-partner-integration).

## Does CleverTap support deep linking?

A. CleverTap supports [deep linking](https://docs.clevertap.com/docs/create-message-push#deep-linkopen-url) that allows you to land the user on a particular application section.

## Is it possible to get the user list in segments with an API call?

A. Yes. You can use CleverTap APIs to push and retrieve data. You can check more details about using CleverTap APIs [here](https://developer.clevertap.com/docs/api-overview).
