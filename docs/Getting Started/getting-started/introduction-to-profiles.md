## Overview

A user profile is automatically created for every user who launches an app or visits a website. User profiles are crucial for creating an effective and strong segmentation to target a set of users with timely, contextual, and personalized campaigns. Over time, segmented campaigns can increase the stickiness of an app or website usage.
