## Overview

This section covers how an admin can grant a passcode after a user has been added to a project. 

# Grant a Passcode

To grant a passcode, perform the following steps:

1. From the dashboard, navigate to *Settings* > *Users*.
2. Click the vertical ellipsis on the appropriate user.
3. Click the lock icon. 

![1600](https://files.readme.io/3758c06-Chunk_1_-_Grant_Passcode_1.png "Chunk 1 - Grant Passcode 1.png")

4. Once the pop-up displays, type a number for the days or select *Forever* to define how long you are granting the passcode for.
5. Click **Grant**.

![1008](https://files.readme.io/723e878-Chunk_1_-_Grant_Passcode_2.png "Chunk 1 - Grant Passcode 2.png")
