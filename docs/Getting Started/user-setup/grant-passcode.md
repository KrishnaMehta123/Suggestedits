# Overview
Granting a user passcode helps authenticate the user when making API calls. For more information about API authentication in CleverTap, refer to [Authentication](https://developer.clevertap.com/docs/authentication).

#Grant a Passcode
To grant a passcode:
1. From the dashboard, navigate to *Settings* > *Users*.
2. Click the ![Ellipsis](https://files.readme.io/2717d2a-ellipses_icon.png) icon against the required user and click the ![Grant passco](https://files.readme.io/cc0d739-Grant_passcode_icon.png) icon.
