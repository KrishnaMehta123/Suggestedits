# Overview

Revoking user access invalidates the existing passcode for the user, and the user can no longer make API calls using their passcode.

# Grant a Passcode

To grant a passcode:

1. From the dashboard, navigate to *Settings* > *Users*.
2. Click the ![Ellipses](https://files.readme.io/2717d2a-ellipses_icon.png) against the required user and click the ![Revoke access](https://files.readme.io/9964f20-Revoke_access_icon.png) icon.\

<Image title="Chunk 1 - Grant Passcode 1.png" alt={1600} className="border" border={true} src="https://files.readme.io/3758c06-Chunk_1_-_Grant_Passcode_1.png" />

4. Click **Revoke** when prompted to confirm the action.\
