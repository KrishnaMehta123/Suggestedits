## Overview

This section covers how an admin can edit a role after a user has been added to a project. 

# Edit a Role

To edit a role, perform the following steps:

1. From the dashboard, navigate to *Settings* > *Users*.
2. Click the vertical ellipsis on the appropriate user.
3. Click the pencil icon.

![1600](https://files.readme.io/6bd6682-Chunk_1_-_Reset_Passcode_1.png "Chunk 1 - Reset Passcode 1.png")

4. Use the *+ Role* dropdown to unselect an existing role, then select a new role.
5. Click **Save**.

![883](https://files.readme.io/b56b3c2-Edit_role_2.png "Edit role 2.png")
