Once a new user has been added to a project, the dashboard provides an admin the ability to view key information about the user. 

#View User Basic Information
To verify the user name, email, role, status, and the passcode-status, perform the following steps:
1. From the dashboard, navigate to *Settings* > *Users*.
2. In the search box, enter the name or user's email.
3. View the user's information.
