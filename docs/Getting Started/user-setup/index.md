## Overview

The account admin can perform various functions to set up a user such as adding users and editing roles. Before making user changes, the admin must invite users to a project to the dashboard and define their role. 

For more information, refer to [Invite Team Members](https://docs.clevertap.com/docs/set-up#section-invite-team-members).
