This section covers how an admin can reset a passcode after a user has been added to a project. 

#Reset a Passcode

To reset a passcode, perform the following steps:
1. From the dashboard, navigate to *Settings* > *Users*.
2. Click the vertical ellipsis on the appropriate user.
3. Click the circular arrows icon.
