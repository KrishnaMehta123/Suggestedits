## Overview

This section covers how to assign user roles in order to manage users within your organization with different access.

# Assign a User Role

To assign a user role, perform the following steps:

1. From the CleverTap dashboard, navigate to *Settings* > *Users*. 
2. Select as many users as required.

![1409](https://files.readme.io/6ff246d-Assign_user_roles_1.png "Assign user roles 1.png")

3. Click **Assign Role** and select a role to assign.
4. Click **Assign**.

![1420](https://files.readme.io/24e5050-Assign_user_roles_2.png "Assign user roles 2.png")
