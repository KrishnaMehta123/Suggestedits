# Sample Custom Events

Below are the essential events that should be tracked on your Content/Media platform to monitor your user interactions and behaviors:

| Event            | Event Property       | Description                                                                                                                                | Data Type |
| :--------------- | :------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- | :-------- |
| Content Viewed   |                      | Captured when content is opened. This allows you to view top content in descending order of views.                                         |           |
|                  | Content Name         | Name of the content (e.g., Money Heist).                                                                                                   |           |
|                  | Content Type         | Type of the content (e.g., Text, Photo, Video).                                                                                            |           |
|                  | Content Category     | Category of the content (e.g., Comedy, Drama, Thriller).                                                                                   |           |
| Content Shared   |                      | Captured when content is shared outside the app. This allows you to analyze the virality of content in the context of social media shares. |           |
|                  | Content Name         | Name of the content.                                                                                                                       |           |
|                  | via Facebook         | Content shared on Facebook.                                                                                                                |           |
|                  | via Twitter          | Content shared on Twitter.                                                                                                                 |           |
| Content Searched |                      | Captured when content is searched. This allows you to track common keywords that have been searched across the website.                    |           |
|                  | Keyword              | Search keyword entered by the user.                                                                                                        |           |
| Commented        |                      | Captured when a user comments on content. This allows you to track interaction in the context of a piece of content.                       |           |
|                  | Content Name         | Name of the content                                                                                                                        |           |
|                  | Author Name          | Name of the Author (e.g., Peter S.)                                                                                                        |           |
| Video Played     |                      | Captured when a Video is played. This allows you to track the most popular videos being viewed.                                            |           |
|                  | Content Name         | Name of the content                                                                                                                        |           |
|                  | Episode/Video Number | Name of the Episode or Video played.                                                                                                       |           |
|                  | Duration             | Time elapsed since the video was played.                                                                                                   |           |
| Subscribe        |                      | Captured when a user takes the subscription.                                                                                               |           |
|                  | Subscription  Type   | Type of subscription (e.g., Gold, Platinum).                                                                                               |           |
