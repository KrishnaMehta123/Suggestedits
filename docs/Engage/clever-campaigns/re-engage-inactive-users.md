# Get Back Non-Returning Users

Use our “Get back non-returning users” campaign to tap new users on Day 20 (or any number of days you select) with a push notification that drives them back into your app if they haven’t returned to your app in that number of days. 

<Image title="clever-campaign-4.png" alt={1403} className="border" border={true} src="https://files.readme.io/521f079-clever-campaign-4.png" />

Click on “Edit Campaign”, then personalize with text and custom user attributes if needed.

<Image title="be1.png" alt={880} className="border" border={true} src="https://files.readme.io/32632cf-be1.png" />

After clicking “Next >”, you can add a deep link so that clicking on this push notification will directly open that link for the user, and add an image to the push notification as well, as follows.

<Image title="be2.png" alt={880} className="border" border={true} src="https://files.readme.io/5c66a7f-be2.png" />

# Send App Version Upgrade Reminders

Use our “App version upgrade reminders” campaign to remind users to download the new version of your app with a push notification and an email. This will happen automatically when CleverTap recognizes that you’ve updated a version of your app. 

<Image title="clever-campaign-5.png" alt={1403} className="border" border={true} src="https://files.readme.io/36ade04-clever-campaign-5.png" />

Click on “Edit Campaign”, then personalize your message with text and custom user attributes if needed.
